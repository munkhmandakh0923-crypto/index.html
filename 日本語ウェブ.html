道具 Дархны Багаж
Ажлын байранд хэрэглэдэг бүх багаж — Япон нэр + Монгол тайлбар. Картыг дарж нэмэлт мэдээлэл харна уу.
Хэрхэн ашиглах вэ? — Картыг дараад ар талыг нь харна уу. Япон нэрийг сайн цээжилж, бригадиртайгаа харилцахдаа ашиглаарай.
この道具は何ですか？ Kono dōgu wa nan desu ka? — Энэ багаж юу вэ?
漢字 Кани
Дархны ажилд зайлшгүй хэрэгтэй кани үсгүүд — тайлбар, жишээтэй
Кани гэж юу вэ? Япон хэлэнд 3 төрлийн үсэг байдаг: ひらがな (хирагана), カタカナ (катакана), 漢字 (кани). Кани нь Хятад гаралтай тэмдэгт бөгөөд нэг тэмдэгт нь бүхэл утгыг илэрхийлдэг. Дархны ажилд дараах кани үсгүүд тохиодоно.
ひらがな Хирагана
Япон хэлний үндсэн цагаан толгой — 46 үсэг, бүгдийг цээжил!
Яагаад хирагана чухал вэ? Хирагана нь үйл үгийн төгсгөл, нөхцөл, холбох үг болгон хэрэглэгддэг. Жишээ нь: 切る、切って、切らない — улаан хэсэг нь бүгд хирагана. Эдгээрийг мэдэхгүй бол үйл үг хувиргах боломжгүй!

助詞 Нөхцөл
Японы нөхцөл = Монгол тийн ялгал! Та аль хэдийн мэдэж байна — зүгээр л шинэ дүрслэл цээжлэх хэрэгтэй.
💡 Монгол тийн ялгалтай харьцуулал
Монгол тийн ялгал Монгол нөхцөл Япон нөхцөл Үүрэг
Нэрийн тийн ялгал — (нөхцөлгүй) は・が Сэдэв ба эзэн заана
Үйлтийн тийн ялгал -ыг / -ийг / -г を Тусагдахуун (юуг хийж байна)
Өгөх-Ортын тийн ялгал -д / -т / -руу / -рүү に・へ Чиглэл, зорилт, байршил
Гарах тийн ялгал -аас / -ээс / -оос から・より Эхлэх цэг, гарал
Хамтын тийн ялгал (-аар) -аар / -ээр / -өөр で Хэрэгсэл, арга зам
Хамтын тийн ялгал (-тай) -тай / -тэй / -той と Хүнтэй хамт
Байрын тийн ялгал -д / -т (байршил) に (оршин) / で (үйлдэл) Байршил — оршин/үйлдэл ялгана
Харьяалах тийн ялгал -ын / -ийн / -н の Өмчлөл, харьяалал
は

≈ Нэрийн тийн ялгал — сэдэв
Өгүүлбэрийн сэдвийг тодорхойлно. "Энэ тухай ярьж байна" гэсэн утга. が-тай ялгаа: は = сэдэв (мэдэгдсэн зүйл), が = шинэ эсвэл онцлон заасан эзэн.
